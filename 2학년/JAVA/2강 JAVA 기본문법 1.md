# JAVA 기본문법

## 식별자
- 프로그래머가 작명하는 이름 (클래스, 변수, 메소드 등의 이름)

### 식별자 작명 규칙
- 대소문자 구분, 길이 제한 없음
- 영어, 한글, 숫자, _, $ 사용 가능 (자바는 유니코드를 사용해서 한글 사용 가능 !)
- 숫자로 시작 불가
- true, false, null, class 등은 사용 불가

**클래스**

- 파스칼 케이스 (UserDto)
- 대문자로 시작하고 연결되는 부분도 대문자

**메소드, 변수**

- 카멜 케이스 (getInfo)
- 소문자로 시작 연결 되는 부분 대문자

**상수**

- 스네이크 케이스 (USER_DIV)
- 전부 대문자이고 연결 되는 부분 언더바(_) 사용

## 자료형
### 변수와 자료형
- 변수 선언 시 저장되는 값의 자료형을 선언한다. (int a)
- 메소드를 선언할 때 파리미터와 반환 값의 자료형을 선언
- 자료형에 따라 적용 가능한 연산이 다르다.

#### 지역변수와 파라미터

선언된 곳부터 해당 블록이 종료될 때까지 유효 -> 메소드가 실행될 때 만들어지고 종료되면 사라짐

- 지역 변수는 초기값 지정 후 사용
- 지역 변수 선언에서 접근 제어자를 사용하지 않음

#### 데이터 필드

클래스 내부에서 사용 가능 (외부 클래스의 사용 가능 여부는 접근 제어자에 따라 다르다.)

- 인스턴스 변수 : non static 필드. 개별 객체가 소유함
- static 변수 : 동종 객체간 공유하는 변수

#### 참조형
주소 값을 나태내는 자료형. 객체의 주소값을 저장한다.
- 실제 객체는 동적 멤모리 공간에 저장한다. (기본형 변수는 저장 공간에 값 자체를 저장.)
- 클래스, 배열, 인터페이스 등등 이 참조형 변수입니둥

### 형변환
#### 묵시적형 변환
작은 타입에서 큰 타입으로의 자동 형변환

byte -> short -> int -> long -> float -> double

#### 명시적 자료형 변환

큰 타입에서 작은 타입으로 변환할 때 -> 강제로 변환

float f = (float)5.5; // 5.5 double이라 자동형변환 되지 않는다
